About Us

The WCU Computer Science club is open to computer science majors and non-majors. We focus on all aspects of computer science, including programming, debates, web design, collaborative projects, information security, and other areas.

We meet weekly in 25 University Avenue, Room 147. We also put on events, such as a programming competition and movie nights.


Programming problem of the week:

A Pythagorean triplet is a set of three natural numbers, a < b < c, for which,
     a2 + b2 = c2.

For example, 32 + 42 = 9 + 16 = 25 = 52.

There exists exactly one Pythagorean triplet for which a + b + c = 1000.
Find the product abc.
